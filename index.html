Standup

Github
Standup allows for collaborative preparation and participation in meetings. Before and during a meeting, all members can contribute to the agenda by posting and voting on topics. After a meeting is closed, all meeting members receive a summary email and access to an archived version of the meeting. We built Standup in eight days using Ruby on Rails for the backend, and jQuery, AJAX, HTML and CSS for the front-end. Our app is tightly coupled with Google (Calendar and Contacts APIs) for user convenience. We accomplished rapid live page updating with Faye Websockets running on a separate Heroku app. We have a Sidekiq worker process and a Redis server to speed up contact caching and emailing (SendGrid)
October 2014

You Owe Me, Homie

Github
You Owe Me Homie lets you track debit and credit between members of a customized group. Recipients must confirm each transaction before the app factors it into calculations. Users can view histories of confirmed, pending, and denied transactions between any two members. We built You Owe Me, Homie in five days using test driven development, Rails for the backend and jQuery, AJAX, HTML and CSS for the front.
September 2014

Menu Combinator

Github
A Sinatra app that accepts a data file depicting menu items, prices, and a target price. Upon upload, the app displays the menu, along with a list of all possible combinations adding up exactly to the target price specified. This was an exercise in NP-Complete algorithms and data validation.
